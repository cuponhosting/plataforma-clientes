plataforma-clientes (Cuponhosting.com)
===================

Plataforma de administración y control de productos y servicios para los clientes de cuponhosting.com.

La aplicación le muestra a los clientes los dominios y los planes de hosting que han adquirido

- Muestra fecha de vencimiento en colores verde, naranja, rojo y negro dependiendo del estado del pruducto/servicio que esta consultado. Si el servicio esta vigente la fecha de vencimiento aparece en verde, si faltan 45 días para vencerse aparece naranja, si faltan 30 días para vencerse aparece rojo y si se venció aparece en negro.
